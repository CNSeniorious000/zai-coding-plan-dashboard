# 入门指南

本指南将帮助您开始使用 Z.AI 使用情况仪表板。

## 前提条件

在开始之前，请确保您拥有：

1. 有效的 Z.AI 编码计划订阅
2. 来自 Z.AI 平台的 API 密钥

## 获取 API 密钥

1. 访问 [Z.AI 平台](https://z.ai/manage-apikey/apikey-list)
2. 登录您的账户
3. 在侧边栏中导航到 **API 密钥**
4. 创建新的 API 密钥或复制现有密钥

<Callout type="warn">
保护好您的 API 密钥！切勿公开分享或提交到版本控制系统。
</Callout>

## 使用仪表板

### 步骤 1：输入 API 密钥

在仪表板主页上，您将看到一个 API 密钥输入框。将您的密钥粘贴到那里。

### 步骤 2：获取使用数据

点击 **获取使用情况** 按钮来检索您的使用统计数据。

### 步骤 3：分析数据

仪表板将显示：

- **配额概览**：显示令牌和 MCP 的当前使用百分比
- **模型使用情况**：按模型分类的令牌使用详细信息
- **工具使用情况**：工具调用和成功率统计
- **可视化图表**：柱状图和饼图便于分析

## 理解指标

### 令牌使用量（5 小时）

显示当前 5 小时周期内的令牌消耗。配额每 5 小时重置一次。

### MCP 使用量（1 个月）

如果适用于您的计划，显示 MCP（模型调用点数）的月度使用情况。

### 模型使用情况

- **输入令牌**：发送到模型的令牌
- **输出令牌**：模型生成的令牌
- **总令牌数**：输入和输出的总和
- **请求次数**：进行的 API 调用次数

## 高效使用技巧

1. **定期监控**：检查使用情况以避免达到限制
2. **优化提示**：使用简洁的提示来减少令牌消耗
3. **选择合适的模型**：对简单任务使用更轻量的模型
