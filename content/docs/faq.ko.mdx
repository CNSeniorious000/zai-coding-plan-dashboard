# 자주 묻는 질문

## 일반

### Z.AI 사용 대시보드란 무엇인가요?

Z.AI 사용 대시보드는 GLM 코딩 플랜 사용량 및 할당량을 모니터링하는 데 도움이 되는 웹 애플리케이션입니다. 토큰 소비의 실시간 통계 및 시각화를 제공합니다.

### API 키가 저장되나요?

아니요, API 키는 Z.AI 서버로의 요청을 만드는 데만 사용되며 당사 서버에는 저장되지 않습니다. 세션 동안 브라우저 메모리에 일시적으로 보관됩니다.

### 할당량은 얼마나 자주 재설정되나요?

토큰 할당량은 5시간마다 재설정됩니다. 정확한 재설정 시간은 현재 주기가 언제 시작되었는지에 따라 달라집니다.

## 사용

### "사용 가능한 데이터가 없습니다"가 표시되는 이유는 무엇인가요?

다음과 같은 경우에 발생할 수 있습니다：
- 선택한 기간 동안 API 호출을하지 않았습니다
- API 키가 유효하지 않거나 만료되었습니다
- Z.AI API에 일시적인 문제가 있습니다

### 할당량에 무엇이 포함되나요?

- GLM 모델과의 모든 채팅 완료
- 입력 토큰과 출력 토큰 모두
- 코딩 어시스턴트 사용 시 도구 호출

### 무료 등급으로 이 대시보드를 사용할 수 있나요?

이 대시보드는 유료 코딩 플랜 구독자를 위해 설계되었습니다. 무료 등급 사용자는 사용 통계에 대한 액세스가 제한될 수 있습니다.

## 기술

### 어떤 브라우저가 지원되나요?

대시보드는 모든 최신 브라우저에서 작동합니다：
- Chrome(권장)
- Firefox
- Safari
- Edge

### 소스 코드를 사용할 수 있나요?

네, 이 프로젝트는 오픈 소스입니다. GitHub에서 코드를 확인하고 기여할 수 있습니다.

### 버그를 보고하거나 기능을 요청하려면 어떻게 하나요?

버그나 기능 요청에 대한 자세한 설명과 함께 GitHub 저장소에서 이슈를 열어주세요.

## 플랜 및 결제

### 플랜을 업그레이드하려면 어떻게 하나요?

1. [Z.AI 구독 페이지](https://z.ai/manage-apikey/subscription)로 이동
2. 원하는 플랜 선택
3. 결제 프로세스 완료

### 어떤 결제 방법이_accepted됩니까?

Z.AI는 주요 신용카드 및 기타 결제 방법을 받습니다. 현재 옵션은 구독 페이지를 확인하세요.

### 환불을 받을 수 있나요?

환불 문의는 Z.AI 지원팀에 직접 문의해 주세요.
