# Tableau de Bord d'Utilisation Z.AI

Bienvenue dans la documentation du Tableau de Bord d'Utilisation Z.AI. Cet outil vous aide à surveiller votre utilisation et quota du Plan de Codage GLM en temps réel.

## Qu'est-ce que le Plan de Codage Z.AI ?

Le Plan de Codage GLM est un package d'abonnement conçu spécifiquement pour le codage IA. Il donne accès à GLM-4.7 et d'autres modèles dans les meilleurs outils de codage comme Claude Code, Cline, OpenCode et plus encore.

### Plans Disponibles

| Plan | Prix | Prompts par 5 heures |
|------|------|---------------------|
| Lite | $3/mois | ~120 prompts |
| Pro | $15/mois | ~600 prompts |
| Max | $30/mois | ~2400 prompts |

## Fonctionnalités

Ce tableau de bord fournit :

- **Surveillance de Quota en Temps Réel** : Voyez votre utilisation actuelle de jetons et le quota restant
- **Statistiques d'Utilisation des Modèles** : Suivez les modèles que vous utilisez et combien de jetons
- **Analyse d'Utilisation des Outils** : Surveillez vos appels d'outils et taux de succès
- **Graphiques Visuels** : De belles visualisations de vos modèles d'utilisation

## Démarrage Rapide

1. Obtenez votre clé API de [Plateforme Z.AI](https://z.ai/manage-apikey/apikey-list)
2. Entrez votre clé API dans le tableau de bord
3. Cliquez sur "Récupérer l'Utilisation" pour voir vos statistiques
