# Z.AI Usage Dashboard

Welcome to the Z.AI Usage Dashboard documentation. This tool helps you monitor your GLM Coding Plan usage and quota in real-time.

## What is Z.AI Coding Plan?

The GLM Coding Plan is a subscription package designed specifically for AI-powered coding. It provides access to GLM-4.7 and other models in top coding tools like Claude Code, Cline, OpenCode, and more.

### Plans Available

| Plan | Price | Prompts per 5 hours |
|------|-------|---------------------|
| Lite | $3/month | ~120 prompts |
| Pro | $15/month | ~600 prompts |
| Max | $30/month | ~2400 prompts |

## Features

This dashboard provides:

- **Real-time Quota Monitoring**: See your current token usage and remaining quota
- **Model Usage Statistics**: Track which models you're using and how many tokens
- **Tool Usage Analytics**: Monitor your tool calls and success rates
- **Visual Charts**: Beautiful visualizations of your usage patterns

## Quick Start

1. Get your API key from [Z.AI Platform](https://z.ai/manage-apikey/apikey-list)
2. Enter your API key in the dashboard
3. Click "Fetch Usage" to view your statistics
