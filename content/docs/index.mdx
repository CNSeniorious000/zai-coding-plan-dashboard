Welcome to the Z.AI Usage Dashboard documentation. This tool helps you monitor your GLM Coding Plan usage and quota in real-time.

## What is Z.AI Coding Plan?

The GLM Coding Plan is a subscription package designed specifically for AI-powered coding. It provides access to GLM-4.7 and other models in top coding tools like Claude Code, Cline, OpenCode, and more.

### Plans Available

| Plan | Price | Prompts per 5 hours |
|------|-------|---------------------|
| Lite | $3/month | ~120 prompts |
| Pro | $15/month | ~600 prompts |
| Max | $30/month | ~2400 prompts |

## Quick Start

<Steps>
<Step>

### Get Your API Key

Visit [Z.AI Platform](https://z.ai/manage-apikey/apikey-list) and copy your API key.

</Step>

<Step>

### Enter Your API Key

Paste your API key in the dashboard input field.

</Step>

<Step>

### View Your Statistics

Click **Fetch Usage** to see your real-time usage data and charts.

</Step>
</Steps>
