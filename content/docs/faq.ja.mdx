# よくある質問

## 一般

### Z.AI 利用状況ダッシュボードとは？

Z.AI 利用状況ダッシュボードは、GLM コーディングプランの使用状況とクォータを監視するのに役立つ Web アプリケーションです。トークン消費のリアルタイム統計と可視化を提供します。

### API キーは保存されますか？

いいえ、API キーは Z.AI サーバーへのリクエストを行うためにのみ使用され、当社のサーバーには保存されません。セッション中は一時的にブラウザのメモリに保存されます。

### クォータはどのくらいの頻度でリセットされますか？

トークンクォータは5時間ごとにリセットされます。正確なリセット時刻は、現在のサイクルがいつ開始されたかによって異なります。

## 使用状況

### 「データがありません」と表示されるのはなぜですか？

これは以下の場合に発生する可能性があります：
- 選択した期間内に API 呼び出しを行っていない
- API キーが無効または期限切れである
- Z.AI API の一時的な問題がある

### 何がクォータにカウントされますか？

- GLM モデルとのすべてのチャット完了
- 入力トークンと出力トークンの両方
- コーディングアシスタント使用時のツール呼び出し

### 無料枠でこのダッシュボードを使用できますか？

このダッシュボードは、有料のコーディングプランサブスクライバー向けに設計されています。無料枠ユーザーは、使用統計へのアクセスが制限される場合があります。

## 技術

### 対応ブラウザは？

ダッシュボードはすべての最新ブラウザで動作します：
- Chrome（推奨）
- Firefox
- Safari
- Edge

### ソースコードは公開されていますか？

はい、このプロジェクトはオープンソースです。GitHub でコードを確認し、貢献できます。

### バグを報告したり、機能をリクエストしたりするにはどうすればよいですか？

バグや機能リクエストの詳細を記載して、GitHub リポジトリでイシューを開いてください。

## プランと請求

### プランをアップグレードするにはどうすればよいですか？

1. [Z.AI サブスクリプションページ](https://z.ai/manage-apikey/subscription)に移動
2. 希望のプランを選択
3. 支払いプロセスを完了

### どのような支払い方法が受け付けられますか？

Z.AI では、主要なクレジットカードやその他の支払い方法を受け付けています。現在のオプションについては、サブスクリプションページをご確認ください。

### 返金を受け取ることはできますか？

返金に関するお問い合わせは、Z.AI サポートまで直接ご連絡ください。
