# Z.AI 사용 대시보드

Z.AI 사용 대시보드 문서에 오신 것을 환영합니다. 이 도구는 GLM 코딩 플랜 사용량 및 할당량을 실시간으로 모니터링하는 데 도움이 됩니다.

## Z.AI 코딩 플랜이란?

GLM 코딩 플랜은 AI 기반 코딩을 위해 특별히 설계된 구독 패키지입니다. Claude Code, Cline, OpenCode 등 최고 코딩 도구에서 GLM-4.7 및 기타 모델에 액세스할 수 있습니다.

### 사용 가능한 플랜

| 플랜 | 가격 | 5시간당 프롬프트 수 |
|------|------|---------------------|
| Lite | $3/월 | 약 120회 |
| Pro | $15/월 | 약 600회 |
| Max | $30/월 | 약 2400회 |

## 기능

이 대시보드는 다음을 제공합니다：

- **실시간 할당량 모니터링**: 현재 토큰 사용량 및 남은 할당량 표시
- **모델 사용 통계**: 사용 중인 모델과 토큰 수 추적
- **도구 사용 분석**: 도구 호출 및 성공률 모니터링
- **시각적 차트**: 사용 패턴의 아름다운 시각화

## 빠른 시작

1. [Z.AI 플랫폼](https://z.ai/manage-apikey/apikey-list)에서 API 키 가져오기
2. 대시보드에 API 키 입력
3. "사용량 가져오기"를 클릭하여 통계 표시
