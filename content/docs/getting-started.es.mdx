# Primeros Pasos

Esta guía te ayudará a comenzar con el Panel de Uso de Z.AI.

## Requisitos Previos

Antes de comenzar, asegúrate de tener：

1. Una suscripción activa del Plan de Codificación Z.AI
2. Tu clave API de la plataforma Z.AI

## Obtener tu Clave API

1. Ve a [Plataforma Z.AI](https://z.ai/manage-apikey/apikey-list)
2. Inicia sesión en tu cuenta
3. Navega a **Claves API** en la barra lateral
4. Crea una nueva clave API o copia una existente

<Callout type="warn">
¡Mantén tu clave API segura! Nunca la compartas públicamente o la envíes a control de versiones.
</Callout>

## Usar el Panel

### Paso 1: Ingresa tu Clave API

En la página de inicio del panel, verás un campo para tu clave API. Pega tu clave allí.

### Paso 2: Obtener Datos de Uso

Haz clic en el botón **Obtener Uso** para recuperar tus estadísticas de uso.

### Paso 3: Analizar tus Datos

El panel mostrará：

- **Resumen de Cuota**: Muestra tu porcentaje actual de uso para tokens y MCP
- **Uso de Modelos**: Desglose detallado del uso de tokens por modelo
- **Uso de Herramientas**: Estadísticas sobre llamadas a herramientas y tasas de éxito
- **Gráficos Visuales**: Gráficos de barras y circulares para análisis fácil

## Entendiendo las Métricas

### Uso de Tokens (5 Horas)

Muestra tu consumo de tokens dentro del ciclo actual de 5 horas. La cuota se restablece cada 5 horas.

### Uso de MCP (1 Mes)

Uso mensual de MCP (Puntos de Llamada de Modelo) si aplica a tu plan.

### Uso de Modelos

- **Tokens de Entrada**: Tokens enviados al modelo
- **Tokens de Salida**: Tokens generados por el modelo
- **Tokens Totales**: Combinación de entrada y salida
- **Recuento de Solicitudes**: Número de llamadas API realizadas

## Consejos para Uso Eficiente

1. **Monitorear Regularmente**: Revisa tu uso para evitar alcanzar límites
2. **Optimizar Prompts**: Usa prompts concisos para reducir consumo de tokens
3. **Elegir el Modelo Correcto**: Usa modelos más ligeros para tareas más simples
