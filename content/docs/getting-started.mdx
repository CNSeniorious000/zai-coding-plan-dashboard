This guide will help you get started with the Z.AI Usage Dashboard.

## Prerequisites

Before you begin, make sure you have:

- An active Z.AI Coding Plan subscription
- Your API key from the Z.AI platform

<Callout type="info">
Don't have a subscription yet? Visit [Z.AI Pricing](https://z.ai/pricing) to choose a plan that fits your needs.
</Callout>

## Getting Your API Key

<Steps>
<Step>

### Visit Z.AI Platform

Go to [Z.AI Platform](https://z.ai/manage-apikey/apikey-list) and log in to your account.

</Step>

<Step>

### Navigate to API Keys

Click on **API Keys** in the sidebar menu.

</Step>

<Step>

### Create or Copy Key

Create a new API key or copy an existing one. Your key format should be:

```
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx.xxxxxxxxxxxxxxxx
```

</Step>
</Steps>

<Callout type="warn">
Keep your API key secure! Never share it publicly or commit it to version control.
</Callout>

## Using the Dashboard

<Steps>
<Step>

### Enter Your API Key

On the dashboard homepage, you'll see an input field for your API key. You can:
- Paste it manually
- Click the clipboard icon to paste from clipboard

</Step>

<Step>

### Fetch Usage Data

Click the **Fetch Usage** button to retrieve your usage statistics.

</Step>

<Step>

### Analyze Your Data

The dashboard will display your usage data in multiple views.

</Step>
</Steps>

## Understanding the Metrics

<Tabs items={['Token Usage', 'MCP Usage', 'Model Stats']}>
<Tab value="Token Usage">

### Token Usage (5 Hour)

This shows your token consumption within the current 5-hour cycle.

- **Current Usage**: Tokens consumed so far
- **Total Quota**: Your plan's token limit
- **Remaining**: Tokens still available
- **Reset Time**: When the quota resets

</Tab>
<Tab value="MCP Usage">

### MCP Usage (1 Month)

Monthly usage of MCP (Model Call Points) if applicable to your plan.

- Tracks overall API call volume
- Resets at the start of each billing cycle

</Tab>
<Tab value="Model Stats">

### Model Usage Statistics

- **Input Tokens**: Tokens sent to the model
- **Output Tokens**: Tokens generated by the model
- **Total Tokens**: Combined input and output
- **Request Count**: Number of API calls made

</Tab>
</Tabs>
