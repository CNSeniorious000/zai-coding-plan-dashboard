# Getting Started

This guide will help you get started with the Z.AI Usage Dashboard.

## Prerequisites

Before you begin, make sure you have:

1. An active Z.AI Coding Plan subscription
2. Your API key from the Z.AI platform

## Getting Your API Key

1. Go to [Z.AI Platform](https://z.ai/manage-apikey/apikey-list)
2. Log in to your account
3. Navigate to **API Keys** in the sidebar
4. Create a new API key or copy an existing one

<Callout type="warn">
Keep your API key secure! Never share it publicly or commit it to version control.
</Callout>

## Using the Dashboard

### Step 1: Enter Your API Key

On the dashboard homepage, you'll see an input field for your API key. Paste your key there.

### Step 2: Fetch Usage Data

Click the **Fetch Usage** button to retrieve your usage statistics.

### Step 3: Analyze Your Data

The dashboard will display:

- **Quota Overview**: Shows your current usage percentage for tokens and MCP
- **Model Usage**: Detailed breakdown of token usage by model
- **Tool Usage**: Statistics on tool calls and success rates
- **Visual Charts**: Bar charts and pie charts for easy analysis

## Understanding the Metrics

### Token Usage (5 Hour)

This shows your token consumption within the current 5-hour cycle. The quota resets every 5 hours.

### MCP Usage (1 Month)

Monthly usage of MCP (Model Call Points) if applicable to your plan.

### Model Usage

- **Input Tokens**: Tokens sent to the model
- **Output Tokens**: Tokens generated by the model
- **Total Tokens**: Combined input and output
- **Request Count**: Number of API calls made

## Tips for Efficient Usage

1. **Monitor Regularly**: Check your usage to avoid hitting limits
2. **Optimize Prompts**: Use concise prompts to reduce token consumption
3. **Choose Right Model**: Use lighter models for simpler tasks
