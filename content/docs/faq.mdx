## General

<Accordions>
<Accordion title="What is the Z.AI Usage Dashboard?">

The Z.AI Usage Dashboard is a web application that helps you monitor your GLM Coding Plan usage and quota. It provides real-time statistics and visualizations of your token consumption.

**Key features:**
- Real-time quota monitoring
- Model usage breakdown
- Tool call analytics
- Visual charts and graphs

</Accordion>

<Accordion title="Is my API key stored?">

<Callout type="info">
Your privacy is protected.
</Callout>

No, your API key is only used to make requests to Z.AI servers and is **not stored on our servers**. It's kept temporarily in your browser's `localStorage` for convenience.

</Accordion>

<Accordion title="How often does the quota reset?">

The token quota resets every **5 hours**. The exact reset time depends on when your current cycle started. You can see the reset time displayed on the dashboard.

</Accordion>
</Accordions>

## Usage

<Accordions>
<Accordion title='Why am I seeing "No data available"?'>

This could happen if:

1. You haven't made any API calls in the selected time period
2. Your API key is invalid or expired
3. There's a temporary issue with the Z.AI API

**Try these solutions:**
- Verify your API key is correct
- Check if you have an active subscription
- Wait a few minutes and try again

</Accordion>

<Accordion title="What counts towards my quota?">

The following activities count towards your token quota:

- All chat completions with GLM models
- Both input and output tokens
- Tool calls when using coding assistants
- System prompts and context

</Accordion>

<Accordion title="Can I use this dashboard with the free tier?">

This dashboard is designed for **paid Coding Plan subscribers**. Free tier users may have limited access to usage statistics through the API.

</Accordion>
</Accordions>
