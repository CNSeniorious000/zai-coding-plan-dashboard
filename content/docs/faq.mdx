# Frequently Asked Questions

## General

### What is the Z.AI Usage Dashboard?

The Z.AI Usage Dashboard is a web application that helps you monitor your GLM Coding Plan usage and quota. It provides real-time statistics and visualizations of your token consumption.

### Is my API key stored?

No, your API key is only used to make requests to Z.AI servers and is not stored on our servers. It's kept temporarily in your browser's memory during your session.

### How often does the quota reset?

The token quota resets every 5 hours. The exact reset time depends on when your current cycle started.

## Usage

### Why am I seeing "No data available"?

This could happen if:
- You haven't made any API calls in the selected time period
- Your API key is invalid or expired
- There's a temporary issue with the Z.AI API

### What counts towards my quota?

- All chat completions with GLM models
- Both input and output tokens
- Tool calls when using coding assistants

### Can I use this dashboard with the free tier?

This dashboard is designed for paid Coding Plan subscribers. Free tier users may have limited access to usage statistics.

## Technical

### What browsers are supported?

The dashboard works on all modern browsers:
- Chrome (recommended)
- Firefox
- Safari
- Edge

### Is the source code available?

Yes, this project is open source. You can find the code and contribute on GitHub.

### How can I report bugs or request features?

Please open an issue on our GitHub repository with a detailed description of the bug or feature request.

## Plans & Billing

3. Complete the payment process
