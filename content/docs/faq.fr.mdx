# Questions Fréquentes

## Général

### Qu'est-ce que le Tableau de Bord d'Utilisation Z.AI ?

Le Tableau de Bord d'Utilisation Z.AI est une application web qui vous aide à surveiller votre utilisation et quota du Plan de Codage GLM. Il fournit des statistiques en temps réel et des visualisations de votre consommation de jetons.

### Ma clé API est-elle stockée ?

Non, votre clé API n'est utilisée que pour faire des requêtes aux serveurs Z.AI et n'est pas stockée sur nos serveurs. Elle est conservée temporairement dans la mémoire de votre navigateur pendant votre session.

### À quelle fréquence le quota se réinitialise-t-il ?

Le quota de jetons se réinitialise toutes les 5 heures. L'heure exacte de réinitialisation dépend du moment où votre cycle actuel a commencé.

## Utilisation

### Pourquoi vois-je "Aucune donnée disponible" ?

Cela pourrait arriver si：
- Vous n'avez pas fait d'appels API dans la période de temps sélectionnée
- Votre clé API n'est pas valide ou a expiré
- Il y a un problème temporaire avec l'API Z.AI

### Qu'est-ce qui compte vers mon quota ?

- Toutes les complétions de chat avec les modèles GLM
- Les jetons d'entrée et de sortie
- Les appels d'outils lors de l'utilisation des assistants de codage

### Puis-je utiliser ce tableau de bord avec le niveau gratuit ?

Ce tableau de bord est conçu pour les abonnés payants du Plan de Codage. Les utilisateurs du niveau gratuit peuvent avoir un accès limité aux statistiques d'utilisation.

## Technique

### Quels navigateurs sont pris en charge ?

Le tableau de bord fonctionne sur tous les navigateurs modernes：
- Chrome (recommandé)
- Firefox
- Safari
- Edge

### Le code source est-il disponible？

Oui, ce projet est open source. Vous pouvez trouver le code et contribuer sur GitHub.

### Comment puis-je signaler des bugs ou demander des fonctionnalités ?

Veuillez ouvrir un issue sur notre dépôt GitHub avec une description détaillée du bug ou de la demande de fonctionnalité.

## Plans et Facturation

### Comment mettre à niveau mon plan ?

1. Allez sur [Page d'Abonnement Z.AI](https://z.ai/manage-apikey/subscription)
2. Sélectionnez le plan souhaité
3. Complétez le processus de paiement

### Quels modes de paiement sont acceptés？

Z.AI accepte les principales cartes de crédit et autres modes de paiement. Consultez la page d'abonnement pour les options actuelles.

### Puis-je obtenir un remboursement？

Veuillez contacter directement le support Z.AI pour les demandes de remboursement.
