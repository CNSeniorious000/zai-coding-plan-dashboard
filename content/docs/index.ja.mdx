# Z.AI 利用状況ダッシュボード

Z.AI 利用状況ダッシュボードのドキュメントへようこそ。このツールは、GLM コーディングプランの使用状況とクォータをリアルタイムで監視するのに役立ちます。

## Z.AI コーディングプランとは？

GLM コーディングプランは、AI コーディング専用に設計されたサブスクリプションパッケージです。Claude Code、Cline、OpenCode などのトップコーディングツールで GLM-4.7 やその他のモデルにアクセスできます。

### 利用可能なプラン

| プラン | 価格 | 5時間あたりのプロンプト数 |
|--------|------|---------------------------|
| Lite | $3/月 | 約120回 |
| Pro | $15/月 | 約600回 |
| Max | $30/月 | 約2400回 |

## 機能

このダッシュボードは以下を提供します：

- **リアルタイムクォータ監視**: 現在のトークン使用量と残りクォータを表示
- **モデル使用統計**: 使用中のモデルとトークン数を追跡
- **ツール使用分析**: ツール呼び出しと成功率を監視
- **視覚的なチャート**: 使用パターンの美しい可視化

## クイックスタート

1. [Z.AI プラットフォーム](https://z.ai/manage-apikey/apikey-list)から API キーを取得
2. ダッシュボードに API キーを入力
3. 「利用状況を取得」をクリックして統計を表示
