# Häufig Gestellte Fragen

## Allgemeines

### Was ist das Z.AI Nutzungs-Dashboard?

Das Z.AI Nutzungs-Dashboard ist eine Webanwendung, die Ihnen hilft, Ihre GLM-Coding-Plan-Nutzung und Ihr Kontingent zu überwachen. Es bietet Echtzeit-Statistiken und Visualisierungen Ihres Token-Verbrauchs.

### Wird mein API-Schlüssel gespeichert?

Nein, Ihr API-Schlüssel wird nur für Anfragen an Z.AI-Server verwendet und nicht auf unseren Servern gespeichert. Er wird während Ihrer Sitzung vorübergehend im Speicher Ihres Browsers aufbewahrt.

### Wie oft wird das Kontingent zurückgesetzt?

Das Token-Kontingent wird alle 5 Stunden zurückgesetzt. Der genaue Rücksetzzeitpunkt hängt davon ab, wann Ihr aktueller Zyklus begonnen hat.

## Nutzung

### Warum sehe ich "Keine Daten verfügbar"?

Dies kann passieren, wenn：
- Sie im ausgewählten Zeitraum keine API-Aufrufe getätigt haben
- Ihr API-Schlüssel ungültig oder abgelaufen ist
- Ein temporäres Problem mit der Z.AI API vorliegt

### Was zählt zu meinem Kontingent?

- Alle Chat-Vervollständigungen mit GLM-Modellen
- Sowohl Eingabe- als auch Ausgabe-Tokens
- Tool-Aufrufe bei Verwendung von KI-Assistenten

### Kann ich dieses Dashboard mit der kostenlosen Stufe verwenden?

Dieses Dashboard ist für zahlende Coding-Plan-Abonnenten konzipiert. Benutzer der kostenlosen Stufe möglicherweise nur eingeschränkten Zugang zu Nutzungsstatistiken haben.

## Technisch

### Welche Browser werden unterstützt？

Das Dashboard funktioniert auf allen modernen Browsern：
- Chrome (empfohlen)
- Firefox
- Safari
- Edge

### Ist der Quellcode verfügbar？

Ja, dieses Projekt ist Open Source. Sie können den Code finden und auf GitHub beitragen.

### Wie kann ich Bugs melden oder Funktionen anfordern？

Bitte öffnen Sie ein Issue in unserem GitHub-Repository mit einer detaillierten Beschreibung des Bugs oder der Funktionsanfrage.

## Pläne & Abrechnung

### Wie aktualisiere ich meinen Plan？

1. Gehen Sie zu [Z.AI Abonnement-Seite](https://z.ai/manage-apikey/subscription)
2. Wählen Sie den gewünschten Plan
3. Vervollständigen Sie den Zahlungsprozess

### Welche Zahlungsmethoden werden akzeptiert？

Z.AI akzeptiert wichtige Kreditkarten und andere Zahlungsmethoden. Überprüfen Sie die Abonnement-Seite für aktuelle Optionen.

### Kann ich eine Rückerstattung erhalten？

Bitte kontaktieren Sie direkt den Z.AI-Support für Rückerstattungsanfragen.
