# Pour Commencer

Ce guide vous aidera à démarrer avec le Tableau de Bord d'Utilisation Z.AI.

## Prérequis

Avant de commencer, assurez-vous d'avoir：

1. Un abonnement actif au Plan de Codage Z.AI
2. Votre clé API de la plateforme Z.AI

## Obtenir votre Clé API

1. Allez sur [Plateforme Z.AI](https://z.ai/manage-apikey/apikey-list)
2. Connectez-vous à votre compte
3. Naviguez vers **Clés API** dans la barre latérale
4. Créez une nouvelle clé API ou copiez une existante

<Callout type="warn">
Gardez votre clé API en sécurité ! Ne la partagez jamais publiquement ou ne la commitez pas dans le contrôle de version.
</Callout>

## Utiliser le Tableau de Bord

### Étape 1 : Entrez votre Clé API

Sur la page d'accueil du tableau de bord, vous verrez un champ pour votre clé API. Collez votre clé là.

### Étape 2 : Récupérer les Données d'Utilisation

Cliquez sur le bouton **Récupérer l'Utilisation** pour récupérer vos statistiques d'utilisation.

### Étape 3 : Analyser vos Données

Le tableau de bord affichera：

- **Aperçu du Quota** : Montre votre pourcentage actuel d'utilisation pour les jetons et MCP
- **Utilisation des Modèles** : Détaillage de l'utilisation des jetons par modèle
- **Utilisation des Outils** : Statistiques sur les appels d'outils et taux de succès
- **Graphiques Visuels** : Graphiques à barres et circulaires pour une analyse facile

## Comprendre les Métriques

### Utilisation de Jetons (5 Heures)

Montre votre consommation de jetons dans le cycle actuel de 5 heures. Le quota se réinitialise toutes les 5 heures.

### Utilisation MCP (1 Mois)

Utilisation mensuelle des MCP (Points d'Appel de Modèle) si applicable à votre plan.

### Utilisation des Modèles

- **Jetons d'Entrée** : Jetons envoyés au modèle
- **Jetons de Sortie** : Jetons générés par le modèle
- **Jetons Totaux** : Combinaison d'entrée et sortie
- **Nombre de Requêtes** : Nombre d'appels API effectués

## Conseils pour une Utilisation Efficace

1. **Surveiller Régulièrement** : Vérifiez votre utilisation pour éviter d'atteindre les limites
2. **Optimiser les Prompts** : Utilisez des prompts concis pour réduire la consommation de jetons
3. **Choisir le Bon Modèle** : Utilisez des modèles plus légers pour des tâches plus simples
