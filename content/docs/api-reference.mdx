This page documents the Z.AI API endpoints used by the dashboard.

## Authentication

All API requests require authentication using a Bearer token:

```bash
Authorization: Bearer YOUR_API_KEY
```

<Callout type="warn">
Keep your API key secure and never expose it in client-side code.
</Callout>

## Base URL

```
https://api.z.ai
```

## Endpoints

<Tabs items={['Model Usage', 'Tool Usage', 'Quota Limit']}>
<Tab value="Model Usage">

### Get Model Usage

Retrieve model usage statistics for a time period.

```http
GET /api/monitor/usage/model-usage
```

**Query Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| startTime | string | Start time (format: `YYYY-MM-DD HH:mm:ss`) |
| endTime | string | End time (format: `YYYY-MM-DD HH:mm:ss`) |

**Response:**

```json
{
  "data": [
    {
      "model": "glm-4.7",
      "inputTokens": 12345,
      "outputTokens": 6789,
      "totalTokens": 19134,
      "requestCount": 42
    }
  ]
}
```

</Tab>
<Tab value="Tool Usage">

### Get Tool Usage

Retrieve tool usage statistics.

```http
GET /api/monitor/usage/tool-usage
```

**Query Parameters:**

Same as Model Usage endpoint.

**Response:**

```json
{
  "data": [
    {
      "tool": "code_execution",
      "callCount": 100,
      "successCount": 95,
      "failureCount": 5
    }
  ]
}
```

</Tab>
<Tab value="Quota Limit">

### Get Quota Limit

Retrieve current quota and usage limits.

```http
GET /api/monitor/usage/quota/limit
```

**Response:**

```json
{
  "data": {
    "limits": [
      {
        "type": "TOKENS_LIMIT",
        "percentage": 45.5
      },
      {
        "type": "TIME_LIMIT",
        "percentage": 20.0,
        "currentValue": 200,
        "usage": 1000
      }
    ]
  }
}
```

</Tab>
</Tabs>

## Error Handling

The API returns standard HTTP status codes:

| Code | Description | Action |
|------|-------------|--------|
| 200 | Success | Request completed successfully |
| 400 | Bad Request | Check your request parameters |
| 401 | Unauthorized | Verify your API key |
| 429 | Rate Limited | Wait before retrying |
| 500 | Server Error | Try again later |

<Callout type="error">
If you receive a 401 error, your API key may be invalid or expired. Generate a new one from the Z.AI platform.
</Callout>

## Rate Limits

API calls are subject to rate limiting. If you exceed the limit, you'll receive a 429 response.

<Callout type="info">
The dashboard automatically handles rate limiting by spacing out requests. Manual API usage should implement exponential backoff.
</Callout>
