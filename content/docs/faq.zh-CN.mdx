## 常规

<Accordions>
<Accordion title="什么是 Z.AI 使用仪表板？">

Z.AI 使用仪表板是一个 Web 应用程序，帮助您监控 GLM Coding Plan 的使用情况和配额。它提供实时的令牌消耗统计和可视化。

**主要功能：**
- 实时配额监控
- 模型使用情况分析
- 工具调用分析
- 可视化图表

</Accordion>

<Accordion title="我的 API 密钥会被存储吗？">

<Callout type="info">
您的隐私受到保护。
</Callout>

不会，您的 API 密钥仅用于向 Z.AI 服务器发出请求，**不会存储在我们的服务器上**。它会临时保存在您浏览器的 `localStorage` 中以方便使用。

</Accordion>

<Accordion title="配额多久重置一次？">

令牌配额每 **5 小时**重置一次。具体的重置时间取决于您当前周期的开始时间。您可以在仪表板上看到重置时间。

</Accordion>
</Accordions>

## 使用

<Accordions>
<Accordion title='为什么我看到"没有可用数据"？'>

这可能是因为：

1. 在所选时间段内您没有进行任何 API 调用
2. 您的 API 密钥无效或已过期
3. Z.AI API 出现临时问题

**尝试这些解决方案：**
- 验证您的 API 密钥是否正确
- 检查您是否有有效订阅
- 等待几分钟后重试

</Accordion>

<Accordion title="什么会计入我的配额？">

以下活动会计入您的令牌配额：

- 使用 GLM 模型的所有聊天完成
- 输入和输出令牌
- 使用编码助手时的工具调用
- 系统提示和上下文

</Accordion>

<Accordion title="我可以在免费套餐中使用此仪表板吗？">

此仪表板专为**付费 Coding Plan 订阅者**设计。免费套餐用户可能通过 API 访问有限的使用统计信息。

</Accordion>
</Accordions>
