# Z.AI 使用情况仪表板

欢迎使用 Z.AI 使用情况仪表板文档。此工具帮助您实时监控 GLM 编码计划的使用情况和配额。

## 什么是 Z.AI 编码计划？

GLM 编码计划是专为 AI 辅助编程设计的订阅套餐。它在 Claude Code、Cline、OpenCode 等顶级编码工具中提供对 GLM-4.7 和其他模型的访问。

### 可用计划

| 计划 | 价格 | 每 5 小时提示数 |
|------|-------|---------------------|
| Lite | $3/月 | ~120 次 |
| Pro | $15/月 | ~600 次 |
| Max | $30/月 | ~2400 次 |

## 功能特性

此仪表板提供：

- **实时配额监控**：查看当前令牌使用量和剩余配额
- **模型使用统计**：跟踪您正在使用的模型和令牌数量
- **工具使用分析**：监控工具调用和成功率
- **可视化图表**：美观的使用模式可视化

## 快速开始

1. 从 [Z.AI 平台](https://z.ai/manage-apikey/apikey-list) 获取您的 API 密钥
2. 在仪表板中输入您的 API 密钥
3. 点击"获取使用情况"查看您的统计数据
