# Preguntas Frecuentes

## General

### ¿Qué es el Panel de Uso de Z.AI?

El Panel de Uso de Z.AI es una aplicación web que te ayuda a monitorear tu uso y cuota del Plan de Codificación GLM. Proporciona estadísticas en tiempo real y visualizaciones de tu consumo de tokens.

### ¿Se almacena mi clave API?

No, tu clave API solo se usa para hacer solicitudes a los servidores de Z.AI y no se almacena en nuestros servidores. Se mantiene temporalmente en la memoria de tu navegador durante tu sesión.

### ¿Con qué frecuencia se restablece la cuota?

La cuota de tokens se restablece cada 5 horas. La hora exacta de restablecimiento depende de cuándo comenzó tu ciclo actual.

## Uso

### ¿Por qué veo "No hay datos disponibles"?

Esto podría suceder si：
- No has realizado llamadas API en el período de tiempo seleccionado
- Tu clave API no es válida o ha expirado
- Hay un problema temporal con la API de Z.AI

### ¿Qué cuenta hacia mi cuota?

- Todos los completamientos de chat con modelos GLM
- Tanto los tokens de entrada como de salida
- Llamadas a herramientas al usar asistentes de codificación

### ¿Puedo usar este panel con el nivel gratuito?

Este panel está diseñado para suscriptores de pago del Plan de Codificación. Los usuarios del nivel gratuito pueden tener acceso limitado a las estadísticas de uso.

## Técnico

### ¿Qué navegadores son compatibles?

El panel funciona en todos los navegadores modernos：
- Chrome (recomendado)
- Firefox
- Safari
- Edge

### ¿Está disponible el código fuente?

Sí, este proyecto es de código abierto. Puedes encontrar el código y contribuir en GitHub.

### ¿Cómo puedo reportar errores o solicitar características?

Por favor abre un issue en nuestro repositorio de GitHub con una descripción detallada del error o solicitud de característica.

## Planes y Facturación

### ¿Cómo actualizo mi plan?

1. Ve a [Página de Suscripción de Z.AI](https://z.ai/manage-apikey/subscription)
2. Selecciona el plan deseado
3. Completa el proceso de pago

### ¿Qué métodos de pago se aceptan?

Z.AI acepta las principales tarjetas de crédito y otros métodos de pago. Consulta la página de suscripción para las opciones actuales.

### ¿Puedo obtener un reembolso?

Por favor contacta directamente al soporte de Z.AI para consultas de reembolso.
