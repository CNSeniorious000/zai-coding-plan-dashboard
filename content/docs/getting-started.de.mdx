# Erste Schritte

Dieser Leitfaden hilft Ihnen beim Start mit dem Z.AI Nutzungs-Dashboard.

## Voraussetzungen

Stellen Sie vor dem Start sicher, dass Sie haben：

1. Ein aktives Z.AI Coding-Plan-Abonnement
2. Ihren API-Schlüssel von der Z.AI-Plattform

## API-Schlüssel erhalten

1. Gehen Sie zu [Z.AI Plattform](https://z.ai/manage-apikey/apikey-list)
2. Melden Sie sich in Ihrem Konto an
3. Navigieren Sie zu **API-Schlüssel** in der Seitenleiste
4. Erstellen Sie einen neuen API-Schlüssel oder kopieren Sie einen vorhandenen

<Callout type="warn">
Bewahren Sie Ihren API-Schlüssel sicher auf! Geben Sie ihn niemals öffentlich weiter oder committen Sie ihn in die Versionsverwaltung.
</Callout>

## Dashboard verwenden

### Schritt 1: API-Schlüssel eingeben

Auf der Dashboard-Startseite sehen Sie ein Eingabefeld für Ihren API-Schlüssel. Fügen Sie Ihren Schlüssel dort ein.

### Schritt 2: Nutzungsdaten abrufen

Klicken Sie auf die Schaltfläche **"Nutzung abrufen"**, um Ihre Nutzungsstatistiken abzurufen.

### Schritt 3: Daten analysieren

Das Dashboard zeigt：

- **Kontingentübersicht**: Zeigt Ihren aktuellen Nutzungsprozentsatz für Tokens und MCP
- **Modellnutzung**: Detaillierter Aufschlüsselung der Token-Nutzung nach Modell
- **Tool-Nutzung**: Statistiken über Tool-Aufrufe und Erfolgsraten
- **Visuelle Diagramme**: Balken- und Kreisdiagramme für einfache Analyse

## Metriken verstehen

### Token-Nutzung (5 Stunden)

Zeigt Ihren Token-Verbrauch innerhalb des aktuellen 5-Stunden-Zyklus. Das Kontingent wird alle 5 Stunden zurückgesetzt.

### MCP-Nutzung (1 Monat)

Monatliche Nutzung von MCP (Model-Call-Points), falls auf Ihren Plan anwendbar.

### Modellnutzung

- **Input-Tokens**: An das Modell gesendete Tokens
- **Output-Tokens**: Vom Modell generierte Tokens
- **Gesamt-Tokens**: Kombination aus Input und Output
- **Anzahl der Anfragen**: Anzahl der durchgeführten API-Aufrufe

## Tipps für effiziente Nutzung

1. **Regelmäßig überwachen**: Überprüfen Sie Ihre Nutzung, um Limits zu vermeiden
2. **Prompts optimieren**: Verwenden Sie prägnante Prompts, um den Token-Verbrauch zu reduzieren
3. **Richtiges Modell wählen**: Verwenden Sie leichtere Modelle für einfachere Aufgaben
