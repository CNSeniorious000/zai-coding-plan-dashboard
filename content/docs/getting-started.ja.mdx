# スタートガイド

このガイドは、Z.AI 利用状況ダッシュボードを始めるのに役立ちます。

## 前提条件

始める前に、以下があることを確認してください：

1. 有効な Z.AI コーディングプランのサブスクリプション
2. Z.AI プラットフォームの API キー

## API キーの取得

1. [Z.AI プラットフォーム](https://z.ai/manage-apikey/apikey-list)に移動
2. アカウントにログイン
3. サイドバーの **API Keys** に移動
4. 新しい API キーを作成するか、既存のものをコピー

<Callout type="warn">
API キーは安全に保管してください！公開したり、バージョン管理システムにコミットしないでください。
</Callout>

## ダッシュボードの使用

### ステップ 1: API キーを入力

ダッシュボードのホームページに、API キーの入力フィールドがあります。そこにキーを貼り付けます。

### ステップ 2: 使用状況データを取得

**「利用状況を取得」**ボタンをクリックして使用統計を取得します。

### ステップ 3: データを分析

ダッシュボードには以下が表示されます：

- **クォータ概要**: トークンと MCP の現在の使用率を表示
- **モデル使用状況**: モデル別のトークン使用の詳細内訳
- **ツール使用状況**: ツール呼び出しと成功率の統計
- **視覚的なチャート**: 分析しやすい棒グラフと円グラフ

## 指標の理解

### トークン使用量（5時間）

現在の5時間サイクル内のトークン消費量を示します。クォータは5時間ごとにリセットされます。

### MCP 使用量（1ヶ月）

プランに適用される場合、MCP（モデルコールポイント）の月次使用量。

### モデル使用状況

- **入力トークン**: モデルに送信されたトークン
- **出力トークン**: モデルによって生成されたトークン
- **総トークン**: 入力と出力の合計
- **リクエスト数**: API 呼び出しの回数

## 効率的な使用のヒント

1. **定期的に監視**: 制限に達しないように使用状況を確認
2. **プロンプトを最適化**: トークン消費を減らすには簡潔なプロンプトを使用
3. **適切なモデルを選択**: 単純なタスクには軽量モデルを使用
