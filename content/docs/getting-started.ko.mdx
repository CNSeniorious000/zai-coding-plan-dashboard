# 시작하기

이 가이드는 Z.AI 사용 대시보드를 시작하는 데 도움이 됩니다.

## 전제 조건

시작하기 전에 다음이 있는지 확인하세요：

1. 유효한 Z.AI 코딩 플랜 구독
2. Z.AI 플랫폼의 API 키

## API 키 가져오기

1. [Z.AI 플랫폼](https://z.ai/manage-apikey/apikey-list)으로 이동
2. 계정에 로그인
3. 사이드바에서 **API Keys**로 이동
4. 새 API 키를 만들거나 기존 키 복사

<Callout type="warn">
API 키를 안전하게 보관하세요! 공개적으로 공유하거나 버전 관리 시스템에 커밋하지 마세요.
</Callout>

## 대시보드 사용

### 1단계: API 키 입력

대시보드 홈페이지에 API 키 입력 필드가 있습니다. 거기에 키를 붙여넣으세요.

### 2단계: 사용량 데이터 가져오기

**"사용량 가져오기"** 버튼을 클릭하여 사용 통계를 검색합니다.

### 3단계: 데이터 분석

대시보드에 다음이 표시됩니다：

- **할당량 개요**: 토큰 및 MCP의 현재 사용률 표시
- **모델 사용량**: 모델별 토큰 사용의 상세 내역
- **도구 사용량**: 도구 호출 및 성공률 통계
- **시각적 차트**: 분석하기 쉬운 막대형 차트 및 원형 차트

## 지표 이해하기

### 토큰 사용량 (5시간)

현재 5시간 주기 내의 토큰 소비량을 표시합니다. 할당량은 5시간마다 재설정됩니다.

### MCP 사용량 (1개월)

해당되는 경우 MCP(모델 호출 포인트)의 월간 사용량.

### 모델 사용량

- **입력 토큰**: 모델로 전송된 토큰
- **출력 토큰**: 모델에 의해 생성된 토큰
- **총 토큰**: 입력 및 출력 결합
- **요청 수**: 수행된 API 호출 수

## 효율적인 사용을 위한 팁

1. **정기적으로 모니터링**: 제한에 도달하지 않도록 사용량 확인
2. **프롬프트 최적화**: 토큰 소비를 줄이려면 간결한 프롬프트 사용
3. **올바른 모델 선택**: 간단한 작업에는 가벼운 모델 사용
